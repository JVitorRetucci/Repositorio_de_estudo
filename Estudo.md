# Reposítório para estudos
## Criando pasta

+ Criar pasta:

Digitar:

```
mkdir <nome-da-pasta>
```

+ Criando arquivos:

Digitar:

```
touch <nome-do-arquivo.extensão>
```

+ Se localizando:

        Comando "ls"
